Description
A visually appealing Spotify clone built using HTML, CSS, and JavaScript. The project replicates the core features of the Spotify interface, including browsing and playing music tracks. This clone showcases front-end development skills with custom styling, responsive design, and dynamic interactions.
Features
🎵 Music Player: Play, pause, and skip tracks.
🎨 Custom UI: A clean, modern interface inspired by Spotify.
📱 Responsive Design: Optimized for mobile, tablet, and desktop devices.
💡 Interactive Features: Song animations, play/pause buttons, and progress bars.

Technologies Used
HTML5 for structure.
CSS3 for responsive design and styling.
JavaScript for interactivity and media control.

Future Enhancements
📝 Playlist feature: Ability to create and save playlists.
🎤 Search functionality: Add a feature to search for tracks.
